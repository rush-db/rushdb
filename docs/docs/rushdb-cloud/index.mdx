# RushDB Cloud

RushDB Cloud is the managed way to run RushDB. It provides projects, API keys, team access, usage tracking, and a dashboard while preserving the same graph model and APIs used by self-hosted deployments.

## Choose an Operating Model

| Model               | Graph backend                                    | RushDB operations    | Best fit                                               |
| ------------------- | ------------------------------------------------ | -------------------- | ------------------------------------------------------ |
| **RushDB Cloud**    | RushDB-managed Neo4j                             | Managed by RushDB    | Fastest path from API key to production                |
| **Cloud with BYOC** | Your Neo4j or Aura instance                      | Managed by RushDB    | Keep graph data in your own cloud account              |
| **Self-hosted**     | Your Neo4j, PostgreSQL, or SQLite infrastructure | Managed by your team | Full deployment control and no cloud quota enforcement |
