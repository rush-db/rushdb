# Quick Tutorial

This tutorial will help you get started with RushDB by walking through a simple example of creating and querying a small social network.

## Prerequisites

- An API token (see [Obtaining API Token](../get-started/get-api-token))
- Your preferred programming language: Python, TypeScript/JavaScript, or any HTTP client for REST API

## Step 1: Initialize RushDB

Choose your preferred SDK:

import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';

<Tabs groupId="programming-language">
  <TabItem value="typescript" label="TypeScript" default>
```typescript
import RushDB from '@rushdb/javascript-sdk';

const db = new RushDB({
  apiToken: 'your-api-token'
});
```
  </TabItem>
  <TabItem value="python" label="Python">
```python
from rushdb import RushDB

db = RushDB(api_token='your-api-token')
```
  </TabItem>
  <TabItem value="rest" label="REST API">
```bash
# Set your API token for future requests
export TOKEN="your-api-token"
```
  </TabItem>
</Tabs>

## Step 2: Create Users

Let's create two users in our social network:

<Tabs groupId="programming-language">
  <TabItem value="typescript" label="TypeScript" default>
```typescript
const alice = await db.createRecord({
  name: 'Alice',
  age: 28,
  interests: ['coding', 'hiking']
});
await alice.addLabel('Person');

const bob = await db.createRecord({
  name: 'Bob',
  age: 32,
  interests: ['photography', 'travel']
});
await bob.addLabel('Person');
```
  </TabItem>
  <TabItem value="python" label="Python">
```python
alice = db.create_record({
    'name': 'Alice',
    'age': 28,
    'interests': ['coding', 'hiking']
}).add_label('Person')

bob = db.create_record({
    'name': 'Bob',
    'age': 32,
    'interests': ['photography', 'travel']
}).add_label('Person')
```
  </TabItem>
  <TabItem value="rest" label="REST API">
```bash
# Create Alice
curl -X POST "https://api.rushdb.com/v1/records" \
  -H "Authorization: Bearer $TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "properties": {
      "name": "Alice",
      "age": 28,
      "interests": ["coding", "hiking"]
    }
  }'

# Add label to Alice
curl -X POST "https://api.rushdb.com/v1/records/{alice_id}/labels" \
  -H "Authorization: Bearer $TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"label": "Person"}'

# Repeat for Bob...
```
  </TabItem>
</Tabs>

## Step 3: Create Relationships

Let's make Alice and Bob friends:

<Tabs groupId="programming-language">
  <TabItem value="typescript" label="TypeScript" default>
```typescript
await alice.createRelationshipTo(bob, 'FRIENDS_WITH', {
  since: '2024-01-01'
});
```
  </TabItem>
  <TabItem value="python" label="Python">
```python
alice.create_relationship_to(bob, 'FRIENDS_WITH', {
    'since': '2024-01-01'
})
```
  </TabItem>
  <TabItem value="rest" label="REST API">
```bash
curl -X POST "https://api.rushdb.com/v1/relationships" \
  -H "Authorization: Bearer $TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "from": "{alice_id}",
    "to": "{bob_id}",
    "type": "FRIENDS_WITH",
    "properties": {
      "since": "2024-01-01"
    }
  }'
```
  </TabItem>
</Tabs>

## Step 4: Search Records

Let's find all people who are interested in outdoor activities:

<Tabs groupId="programming-language">
  <TabItem value="typescript" label="TypeScript" default>
```typescript
const outdoorsy = await db.search()
  .withLabel('Person')
  .where('interests', 'CONTAINS', ['hiking', 'travel'])
  .execute();

console.log('Found:', outdoorsy.map(p => p.properties.name));
```
  </TabItem>
  <TabItem value="python" label="Python">
```python
outdoorsy = db.search() \
    .with_label('Person') \
    .where('interests', 'CONTAINS', ['hiking', 'travel']) \
    .execute()

print('Found:', [p.properties['name'] for p in outdoorsy])
```
  </TabItem>
  <TabItem value="rest" label="REST API">
```bash
curl -X POST "https://api.rushdb.com/v1/search" \
  -H "Authorization: Bearer $TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "label": "Person",
    "conditions": [{
      "field": "interests",
      "operator": "CONTAINS",
      "value": ["hiking", "travel"]
    }]
  }'
```
  </TabItem>
</Tabs>
